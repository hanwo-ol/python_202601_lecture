# 첫 시간: 파이썬의 기본 연산자들을 알아보자.

프로그래밍의 기초가 되는 연산자(Operator)의 종류와 사용법입니다.

## 1. 산술 연산자 (Arithmetic Operators)

사칙연산을 포함하여 수치 계산을 수행할 때 사용합니다. 파이썬은 정수와 실수를 구분하지 않고 자연스럽게 계산할 수 있습니다.

*   `+` : 더하기
*   `-` : 빼기
*   `*` : 곱하기
*   `/` : 나누기 (결과가 항상 실수형 float으로 반환됨)
*   `//` : 나눗셈의 몫 (소수점 이하 버림)
*   `%` : 나눗셈의 나머지
*   `**` : 거듭제곱 (제곱)

<details>
<summary>산술 연산자 예제 코드 보기</summary>

<br>

```python
a = 10
b = 3

print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.3333...
print(a // b) # 3 (몫)
print(a % b)  # 1 (나머지)
print(a ** b) # 1000 (10의 3승)
```

</details>

---

## 2. 비교 연산자 (Comparison Operators)

두 값을 비교하여 참(True) 또는 거짓(False)을 반환합니다. 조건문(if)이나 반복문(while)에서 주로 사용됩니다.

*   `==` : 같다 (값이 같으면 True)
*   `!=` : 다르다 (값이 다르면 True)
*   `>` : 크다 (왼쪽 값이 더 크면 True)
*   `<` : 작다 (오른쪽 값이 더 크면 True)
*   `>=` : 크거나 같다
*   `<=` : 작거나 같다

<details>
<summary>비교 연산자 예제 코드 보기</summary>

<br>

```python
x = 5
y = 10

print(x == y) # False
print(x != y) # True
print(x < y)  # True
print(x >= 5) # True
```

</details>

---

## 3. 할당 연산자 (Assignment Operators)

변수에 값을 저장(할당)하거나, 연산과 할당을 동시에 수행할 때 사용합니다.

*   `=` : 왼쪽 변수에 오른쪽 값을 저장
*   `+=` : 현재 변수 값에 값을 더한 후 저장 (예: `a += 1` 은 `a = a + 1`과 동일)
*   `-=` : 값을 뺀 후 저장
*   `*=` : 값을 곱한 후 저장
*   `/=` : 값을 나눈 후 저장

<details>
<summary>할당 연산자 예제 코드 보기</summary>

<br>

```python
num = 100 # 할당
print(num)

num += 10 # num = num + 10
print(num) # 110

num -= 20 # num = num - 20
print(num) # 90
```

</details>

---

## 4. 논리 연산자 (Logical Operators)

여러 개의 조건을 결합하거나 부정할 때 사용합니다.

*   `and` : 논리곱 (두 조건이 모두 참이어야 True)
*   `or` : 논리합 (두 조건 중 하나라도 참이면 True)
*   `not` : 논리 부정 (참이면 False, 거짓이면 True로 반전)

<details>
<summary>논리 연산자 예제 코드 보기</summary>

<br>

```python
a = True
b = False

print(a and b) # False (둘 다 참이 아니므로)
print(a or b)  # True (하나라도 참이므로)
print(not a)   # False (True의 반대)

# 실제 활용 예시
score = 85
print(score >= 80 and score < 90) # True (80점대인지 확인)
```

</details>
