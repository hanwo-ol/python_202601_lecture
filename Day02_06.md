# 첫 시간: 파이썬의 기본 연산자들을 알아보자.

프로그래밍의 기초가 되는 연산자(Operator)의 종류와 사용법입니다.

## 1. 산술 연산자 (Arithmetic Operators)

사칙연산을 포함하여 수치 계산을 수행할 때 사용합니다. 파이썬은 정수와 실수를 구분하지 않고 자연스럽게 계산할 수 있습니다.

*   `+` : 더하기
*   `-` : 빼기
*   `*` : 곱하기
*   `/` : 나누기 (결과가 항상 실수형 float으로 반환됨)
*   `//` : 나눗셈의 몫 (소수점 이하 버림)
*   `%` : 나눗셈의 나머지
*   `**` : 거듭제곱 (제곱)

<details>
<summary>산술 연산자 예제 코드 보기</summary>

<br>

```python
a = 10
b = 3

print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.3333...
print(a // b) # 3 (몫)
print(a % b)  # 1 (나머지)
print(a ** b) # 1000 (10의 3승)
```

</details>

---

## 2. 비교 연산자 (Comparison Operators)

두 값을 비교하여 참(True) 또는 거짓(False)을 반환합니다. 조건문(if)이나 반복문(while)에서 주로 사용됩니다.

*   `==` : 같다 (값이 같으면 True)
*   `!=` : 다르다 (값이 다르면 True)
*   `>` : 크다 (왼쪽 값이 더 크면 True)
*   `<` : 작다 (오른쪽 값이 더 크면 True)
*   `>=` : 크거나 같다
*   `<=` : 작거나 같다

<details>
<summary>비교 연산자 예제 코드 보기</summary>

<br>

```python
x = 5
y = 10

print(x == y) # False
print(x != y) # True
print(x < y)  # True
print(x >= 5) # True
```

</details>

---

## 3. 할당 연산자 (Assignment Operators)

변수에 값을 저장(할당)하거나, 연산과 할당을 동시에 수행할 때 사용합니다.

*   `=` : 왼쪽 변수에 오른쪽 값을 저장
*   `+=` : 현재 변수 값에 값을 더한 후 저장 (예: `a += 1` 은 `a = a + 1`과 동일)
*   `-=` : 값을 뺀 후 저장
*   `*=` : 값을 곱한 후 저장
*   `/=` : 값을 나눈 후 저장

<details>
<summary>할당 연산자 예제 코드 보기</summary>

<br>

```python
num = 100 # 할당
print(num)

num += 10 # num = num + 10
print(num) # 110

num -= 20 # num = num - 20
print(num) # 90
```

</details>

---

## 4. 논리 연산자 (Logical Operators)

여러 개의 조건을 결합하거나 부정할 때 사용합니다.

*   `and` : 논리곱 (두 조건이 모두 참이어야 True)
*   `or` : 논리합 (두 조건 중 하나라도 참이면 True)
*   `not` : 논리 부정 (참이면 False, 거짓이면 True로 반전)

<details>
<summary>논리 연산자 예제 코드 보기</summary>

<br>

```python
a = True
b = False

print(a and b) # False (둘 다 참이 아니므로)
print(a or b)  # True (하나라도 참이므로)
print(not a)   # False (True의 반대)

# 실제 활용 예시
score = 85
print(score >= 80 and score < 90) # True (80점대인지 확인)
```

</details>

# 파이썬 연산자와 반복문 실전 연습

앞서 배운 산술, 비교, 할당, 논리 연산자를 반복문과 함께 사용하는 연습 문제입니다.

## 문제 1: 짝수만 골라서 더하기 (산술 + 할당 연산자)

**목표:** 1부터 20까지의 숫자 중에서 **짝수**인 숫자들의 합계만 구하시오.

**조건:**
1. `for` 반복문을 사용하여 1부터 20까지 반복합니다.
2. 산술 연산자(`%`)를 사용하여 짝수인지 판별합니다.
3. 할당 연산자(`+=`)를 사용하여 합계를 누적합니다.

<details>
<summary>문제 1 정답 코드 보기</summary>

<br>

```python
# 합계를 저장할 변수 초기화
total_sum = 0

# 1부터 20까지 반복 (range의 끝 값은 포함되지 않으므로 21)
for i in range(1, 21):
    # 2로 나눈 나머지가 0이면 짝수 (비교 연산자 == 사용)
    if i % 2 == 0:
        # 짝수일 경우 total_sum에 i를 더함 (할당 연산자 += 사용)
        total_sum += i

print("1부터 20까지 짝수의 합:", total_sum)
# 예상 결과: 110
```

</details>

## 문제 2: 목표 금액 모으기 (비교 + 할당 연산자)

**목표:** 매일 1,000원씩 저금을 합니다. 저금통의 금액이 10,000원을 **넘을 때까지(초과)** 저금을 계속하는 상황을 코드로 작성하시오.

**조건:**
1. `while` 반복문을 사용합니다.
2. 비교 연산자(`<=`)를 사용하여 금액이 10,000원 이하일 동안 반복이 실행되도록 합니다.
3. 반복할 때마다 날짜(`day`)는 1씩 증가하고, 금액(`money`)은 1,000씩 증가합니다.

<details>
<summary>문제 2 정답 코드 보기</summary>

<br>

```python
money = 0
day = 1

# money가 10000보다 작거나 같을 동안 계속 반복
while money <= 10000:
    money += 1000  # 1000원씩 저금
    print(day, "일차 저금액:", money)
    day += 1      # 날짜 카운트

print("최종 저금액:", money)
print("목표 달성 소요 일수:", day - 1, "일")
```

</details>

## 문제 3: 공배수 찾기 (논리 연산자)

**목표:** 1부터 50까지의 숫자 중에서 **3의 배수이면서 동시에 4의 배수**인 숫자를 모두 찾아 출력하시오.

**조건:**
1. `for` 반복문을 사용하여 1부터 50까지 반복합니다.
2. 산술 연산자(`%`)와 비교 연산자(`==`)로 배수를 확인합니다.
3. 논리 연산자(`and`)를 사용하여 두 조건을 모두 만족하는 경우를 찾습니다.

<details>
<summary>문제 3 정답 코드 보기</summary>

<br>

```python
print("3과 4의 공배수 목록:")

for i in range(1, 51):
    # 3의 배수(i % 3 == 0) 그리고 4의 배수(i % 4 == 0)
    if i % 3 == 0 and i % 4 == 0:
        print(i)

# 예상 결과: 12, 24, 36, 48
```

</details>

# 파이썬 데이터 타입(Data Types) 가이드

프로그래밍에서 데이터는 그 성격에 따라 처리되는 방식이 다릅니다. 파이썬이 데이터를 어떻게 구분하고 관리하는지, 그 기본적인 형태(Type)에 대해 학습합니다.

## 1. 기본 자료형 (단일 데이터)

하나의 변수에 하나의 값을 저장하는 가장 기본적인 형태입니다.

### 정수형 (int)
소수점이 없는 정수 데이터를 의미합니다. 양수, 음수, 0을 포함하며 크기에 제한 없이 메모리가 허용하는 한 큰 수를 저장할 수 있습니다.

*   **특징:** 사칙연산의 기본이 됩니다.
*   **작성법:** 숫자만 입력합니다.

```python
num1 = 10
num2 = -5
print(type(num1)) # <class 'int'>
```

### 실수형 (float)
소수점이 포함된 숫자 데이터를 의미합니다. 정수보다 더 정밀한 수치를 표현할 때 사용합니다.

*   **특징:** 정수와 연산 시 결과는 실수형이 됩니다. (예: `3 + 2.0 = 5.0`)
*   **작성법:** 숫자에 소수점(`.`)을 포함합니다.

```python
pi = 3.14
rate = 0.0
print(type(pi)) # <class 'float'>
```

### 문자열 (str)
문자들의 나열을 의미합니다. 텍스트 데이터를 처리할 때 사용합니다.

*   **특징:** 덧셈 기호(`+`)로 문자를 연결하거나, 곱셈 기호(`*`)로 반복할 수 있습니다.
*   **작성법:** 작은따옴표(`'`) 또는 큰따옴표(`"`)로 감쌉니다.

```python
text = "Python"
char = 'A'
print(type(text)) # <class 'str'>
```

### 불리언 (bool)
참(True)과 거짓(False) 단 두 가지 값만을 가지는 자료형입니다.

*   **특징:** 조건문에서 논리적 판단을 할 때 사용됩니다.
*   **작성법:** 첫 글자는 반드시 대문자여야 합니다 (`True`, `False`).

```python
is_active = True
is_completed = False
print(type(is_active)) # <class 'bool'>
```

## 2. 컬렉션 자료형 (묶음 데이터)

여러 개의 데이터를 하나로 묶어서 관리하는 형태입니다. 데이터의 **순서 유무**와 **수정 가능 여부**가 중요한 구분 기준이 됩니다.

### 리스트 (list)
여러 데이터를 순서대로 나열하여 저장하는 자료형입니다. 가장 범용적으로 사용됩니다.

*   **특징:** 데이터의 추가, 수정, 삭제가 자유롭습니다.
*   **순서(Index):** 있음 (0부터 시작)
*   **작성법:** 대괄호 `[]`를 사용합니다.

```python
colors = ["red", "green", "blue"]
colors[0] = "yellow" # 첫 번째 요소를 yellow로 수정 가능
```

### 튜플 (tuple)
리스트와 유사하지만, 한 번 생성하면 내부 데이터를 변경할 수 없는 자료형입니다.

*   **특징:** 데이터를 보호해야 하거나, 수정되면 안 되는 기준값을 다룰 때 사용합니다. 리스트보다 메모리를 적게 사용합니다.
*   **순서(Index):** 있음
*   **작성법:** 소괄호 `()`를 사용합니다.

```python
coordinates = (37.5, 127.0)
# coordinates[0] = 38.0  # 에러 발생 (수정 불가)
```

### 딕셔너리 (dict)
키(Key)와 값(Value)이 한 쌍을 이루는 자료형입니다.

*   **특징:** 순차적인 인덱스(0, 1, 2...) 대신 의미 있는 '키'를 통해 '값'을 검색합니다. 검색 속도가 매우 빠릅니다.
*   **순서:** 파이썬 3.7부터는 입력 순서가 유지되지만, 인덱싱으로 접근하지 않고 키로 접근합니다.
*   **작성법:** 중괄호 `{}`를 사용하며 `Key: Value` 형태를 띱니다.

```python
user = {
    "name": "User1",
    "age": 25,
    "city": "Seoul"
}
print(user["name"]) # "User1" 출력
```

### 집합 (set)
수학의 집합 개념을 구현한 자료형입니다.

*   **특징:** 데이터의 중복을 허용하지 않으며, 저장 순서가 없습니다. 따라서 인덱싱(몇 번째 값 가져오기)이 불가능합니다.
*   **용도:** 중복 제거, 교집합/합집합 등의 연산.
*   **작성법:** 중괄호 `{}`를 사용하지만 키와 값이 아닌 데이터만 나열합니다.

```python
numbers = {1, 2, 2, 3, 3}
print(numbers) # {1, 2, 3} (중복 제거됨)
```

## 3. 요약 및 비교

| 자료형 | 괄호 형태 | 순서(Index) | 수정 가능(Mutable) | 중복 허용 | 주요 용도 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **List** | `[ ]` | O | O | O | 데이터의 목록 관리 |
| **Tuple** | `( )` | O | X | O | 변하지 않는 고정 목록 |
| **Dict** | `{ k:v }` | X (Key로 접근) | O | Key(X), Value(O) | 속성이나 정보의 사전적 정의 |
| **Set** | `{ }` | X | O | X | 중복 제거 및 집합 연산 |

### 팁: 데이터 타입 확인하기
파이썬 내장 함수 `type()`을 사용하면 현재 데이터가 어떤 형태인지 즉시 확인할 수 있습니다.

```python
data = [1, 2, 3]
print(type(data)) # <class 'list'> 가 출력됨
```
