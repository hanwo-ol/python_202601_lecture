# 알고리즘 복습
## 정렬 및 탐색 알고리즘 실습
12명의 학생의 각각의 점수가 있다. 
1. 시험 등수가 3등까지 장학금을 받을 수 있을 때, 장학금을 받기 위한 최소 점수는 몇 점인가?
2. 시험 점수가 90점 이상이면 장학금을 받을 수 있을 때, 장학금을 받을 수 있는 학생은 몇 명인가?

[83, 91, 76, 88, 95, 69, 84, 90, 72, 87, 93, 81]


정렬 및 탐색 알고리즘을 이용해서 문제를 풀어라.
단, 컴퓨팅사고를 통해야 하며, 자연어로 작성하고, 수도 코드를 작성한 다음, 파이썬으로 구현한 후, 최종적으로 함수 형태로 기능을 구현하라

### 조교의 뻘짓

## 0. 컴퓨팅 사고(Computational Thinking) 접근

1.  **분해 (Decomposition):** 문제를 두 개의 작은 문제로 나눕니다.
    *   문제 A: 점수를 순서대로 나열하여 3등의 점수 찾기 (정렬 및 인덱싱).
    *   문제 B: 특정 조건(90점 이상)을 만족하는 데이터의 개수 세기 (탐색 및 카운팅).
2.  **패턴 인식 (Pattern Recognition):**
    *   "3등까지"라는 말은 점수가 높은 순서대로 나열(내림차순 정렬)했을 때 3번째 위치한 값을 의미합니다.
    *   "90점 이상"은 리스트의 모든 요소를 순회하며 조건(`>=`)을 검사하는 선형 탐색 패턴입니다.
3.  **추상화 (Abstraction):**
    *   학생의 이름이나 다른 정보는 불필요하며, 오직 '점수 리스트'만 중요합니다.
    *   장학금 커트라인은 정렬된 리스트의 `index 2` (0부터 시작하므로) 값입니다.
4.  **알고리즘 설계 (Algorithm Design):**
    *   데이터 입력 $\rightarrow$ 내림차순 정렬 $\rightarrow$ 3번째 값 추출 $\rightarrow$ 조건에 맞는 개수 탐색 $\rightarrow$ 결과 출력.

---

## 1. 자연어 알고리즘 (Natural Language)

1.  학생 12명의 점수 리스트를 준비합니다.
2.  **[정렬]** 점수가 높은 학생이 먼저 오도록 리스트를 **내림차순**으로 정렬합니다.
3.  **[문제 1 해결]** 정렬된 리스트에서 0번째, 1번째, 2번째 학생까지가 장학금 대상입니다. 따라서 3등에 해당하는 **2번째 인덱스**의 점수를 확인합니다. 이 점수가 장학금을 받기 위한 최소 점수(커트라인)가 됩니다.
4.  **[문제 2 해결]** 리스트에 있는 모든 점수를 하나씩 확인합니다(**탐색**). 만약 점수가 90점 이상이라면 숫자를 1씩 셉니다.
5.  구한 커트라인 점수와 90점 이상인 학생 수를 결과로 반환합니다.

---

## 2. 수도 코드 (Pseudo Code)

```text
START
    // 데이터 입력
    Scores = [83, 91, 76, 88, 95, 69, 84, 90, 72, 87, 93, 81]

    // 1. 정렬 알고리즘 (내림차순)
    SORT Scores in Descending Order

    // 2. 문제 1: 3등까지 장학금 (최소 점수 찾기)
    // 배열 인덱스는 0, 1, 2 이므로 3등은 index 2
    Min_Scholarship_Score = Scores[2]

    // 3. 문제 2: 90점 이상 학생 수 구하기 (선형 탐색)
    High_Score_Count = 0
    FOR each score IN Scores DO
        IF score >= 90 THEN
            High_Score_Count = High_Score_Count + 1
        END IF
    END FOR

    // 결과 출력
    PRINT "장학금 최소 점수:", Min_Scholarship_Score
    PRINT "90점 이상 학생 수:", High_Score_Count
END
```

## 3. 파이썬 구현 (Python Code)

```python
# 데이터 정의
scores = [83, 91, 76, 88, 95, 69, 84, 90, 72, 87, 93, 81]

# 1. 정렬 (Sorting): 내림차순
# 파이썬의 리스트 내장 함수 sort()는 Timsort 알고리즘(O(N log N))을 사용합니다.
scores.sort(reverse=True)

# 정렬된 결과 확인 (디버깅용)
# [95, 93, 91, 90, 88, 87, 84, 83, 81, 76, 72, 69]
print(f"정렬된 점수: {scores}")

# 2. 문제 1 해결: 3등 점수 찾기 (인덱싱)
# 1등(idx 0), 2등(idx 1), 3등(idx 2)
scholarship_cutoff = scores[2]

# 3. 문제 2 해결: 90점 이상 찾기 (탐색)
count_over_90 = 0
for score in scores:
    if score >= 90:
        count_over_90 += 1

# 결과 출력
print(f"1. 장학금을 받기 위한 최소 점수: {scholarship_cutoff}점")
print(f"2. 90점 이상 학생 수: {count_over_90}명")
```

## 4. 함수 형태 구현 (Final Function)

최종적으로 재사용 가능한 함수 형태로 코드를 구조화합니다.

```python
def analyze_scholarship(score_list, rank_limit, score_limit):
    """
    학생 점수를 분석하여 장학금 커트라인과 우수 학생 수를 반환하는 함수
    
    Args:
        score_list (list): 학생들의 점수 리스트
        rank_limit (int): 장학금을 받는 등수 (예: 3등까지)
        score_limit (int): 우수 학생 기준 점수 (예: 90점 이상)
        
    Returns:
        tuple: (커트라인 점수, 기준 점수 이상 학생 수)
    """
    # 원본 데이터 보존을 위해 복사 후 정렬 (Side Effect 방지)
    sorted_scores = sorted(score_list, reverse=True)
    
    # 1. 장학금 커트라인 구하기 (인덱스는 '등수 - 1')
    # 예외 처리: 학생 수가 등수보다 적을 경우 고려
    if len(sorted_scores) >= rank_limit:
        cutoff_score = sorted_scores[rank_limit - 1]
    else:
        cutoff_score = sorted_scores[-1] # 학생 수가 부족하면 꼴등 점수가 커트라인

    # 2. 기준 점수 이상 학생 수 구하기 (선형 탐색)
    count = 0
    for score in sorted_scores:
        if score >= score_limit:
            count += 1
        else:
            # 내림차순 정렬되어 있으므로, 기준 점수보다 낮아지는 순간
            # 뒤의 데이터는 볼 필요 없이 반복 종료 (효율성 최적화 - Early Exit)
            break
            
    return cutoff_score, count

# --- 실행부 ---
students_scores = [83, 91, 76, 88, 95, 69, 84, 90, 72, 87, 93, 81]

# 함수 호출
cutoff, high_scorers = analyze_scholarship(students_scores, 3, 90)

print(f"=== 결과 확인 ===")
print(f"장학금 수여 가능 최소 점수(3등): {cutoff}점")
print(f"90점 이상 학생 수: {high_scorers}명")
```

### 실행 결과 해석
1.  **정렬 결과:** `[95, 93, 91, 90, 88, 87, 84, 83, 81, 76, 72, 69]`
2.  **문제 1:** 3등(인덱스 2)의 점수는 **91점**입니다.
3.  **문제 2:** 90점 이상인 점수는 `95, 93, 91, 90`으로 총 **4명**입니다.
