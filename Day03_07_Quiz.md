# 리스트 인덱싱 및 슬라이싱 평가 문제지

## Part 1. OX 퀴즈 
다음 설명이 맞으면 O, 틀리면 X를 기입하시오.

1. 파이썬 리스트의 인덱스(Index)는 1부터 시작한다. ( )
2. 길이가 $N$인 리스트의 마지막 요소의 인덱스는 $N$이다. ( )
3. 리스트 `L`에 대하여 `L[-1]`은 리스트의 마지막 요소를 가리킨다. ( )
4. 슬라이싱 `L[start:stop]`을 수행할 때, 인덱스 `stop`에 해당하는 요소는 결과에 포함된다. ( )
5. `L[:3]`은 리스트의 인덱스 0, 1, 2에 해당하는 요소를 추출한다. ( )
6. 리스트의 범위를 벗어난 인덱스로 접근(Indexing)을 시도하면 `IndexError`가 발생한다. ( )
7. 리스트의 범위를 벗어난 구간으로 슬라이싱(Slicing)을 시도하면 `IndexError`가 발생한다. ( )
8. 인덱싱의 반환값은 항상 리스트(List) 자료형이다. ( )
9. 슬라이싱의 반환값은 항상 리스트(List) 자료형이다. ( )
10. `L[:]`은 리스트 `L`의 전체 요소를 복사하여 새로운 리스트를 반환한다. ( )
11. 슬라이싱의 `step` 값으로 음수를 사용할 수 없다. ( )
12. `L[3:3]`과 같이 시작 인덱스와 종료 인덱스가 같으면, 해당 인덱스의 요소 1개를 포함한 리스트가 반환된다. ( )
13. `L[::2]`는 리스트의 처음부터 끝까지 인덱스가 짝수(0, 2, 4...)인 요소들을 추출한다. ( )
14. `L[::-1]`은 원본 리스트의 순서를 영구적으로 뒤집는다(Modify). ( )
15. 문자열(String) 자료형에도 리스트와 동일한 방식의 인덱싱과 슬라이싱을 적용할 수 있다. ( )
