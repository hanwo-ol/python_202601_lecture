# 📝 쥬피터 노트북 마크다운 가이드 (초급~중급)

마크다운(Markdown)은 복잡한 HTML 코드 없이도 텍스트를 스타일링할 수 있는 경량 마크업 언어입니다. 데이터 분석가나 개발자가 문서화 작업을 할 때 필수적인 도구입니다.

> **💡 쥬피터 팁:**
> 셀을 선택한 상태에서 단축키 `M`을 누르면 **마크다운 셀**로, `Y`를 누르면 **코드 셀**로 변경됩니다. (작성 후 `Shift + Enter`로 실행)

---

## 🟢 1단계: 초급 (텍스트 꾸미기 기초)

문서를 작성할 때 가장 빈번하게 사용되는 필수 문법입니다.

### 1. 제목 (Headers)
`#`의 개수로 제목의 크기(계층)를 조절합니다. HTML의 `<h1>` ~ `<h6>` 태그와 같습니다.

**[입력]**
```markdown
# 제목 1 (가장 큼)
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6

```

**[결과]**
(글씨 크기가 점점 작아지는 제목들이 생성됩니다.)
# 제목 1 
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6


---

### 2. 텍스트 강조 (Emphasis)
글씨를 굵게 하거나 기울여서 강조할 때 사용합니다.

**[입력]**
```markdown
*이텔릭체* 또는 _이텔릭체_
**볼드체(굵게)** 또는 __볼드체__
~~취소선~~
```

**[결과]**
* *이텔릭체*
* **볼드체(굵게)**
* ~~취소선~~

---

### 3. 리스트 (Lists)
목록을 작성할 때 사용합니다. 순서가 있는 것과 없는 것으로 나뉩니다.

**[입력]**
```markdown
**순서 없는 목록 (기호 사용):**
- 항목 1
  - 하위 항목 (들여쓰기)
* 항목 2
+ 항목 3

**순서 있는 목록 (숫자 사용):**
1. 첫 번째
2. 두 번째
   1. 하위 첫 번째
```

**[결과]**
*   **순서 없는 목록:**
    *   항목 1
        *   하위 항목
    *   항목 2
*   **순서 있는 목록:**
    1.  첫 번째
    2.  두 번째
        1.  하위 첫 번째

---

### 4. 줄바꿈 (Line Break)
마크다운에서는 엔터를 한 번만 치면 줄바꿈이 되지 않는 경우가 있습니다.

*   **방법 1:** 문장 끝에 공백(스페이스바)을 **2칸 이상** 띄우고 엔터
*   **방법 2:** 문장 사이에 `<br>` 태그 입력 (HTML 활용)

---

## 🟡 2단계: 중급 (구조화 및 시각화)

문서를 더욱 깔끔하게 정리하고, 코드나 링크, 표를 넣는 방법입니다.

### 1. 인용구 (Blockquotes)
문서 내에서 특정 문구를 인용하거나 설명 박스를 만들 때 유용합니다.

**[입력]**
```markdown
> 이것은 인용구입니다.
> > 중첩해서 사용할 수도 있습니다.
```

**[결과]**
> 이것은 인용구입니다.
>> 중첩해서 사용할 수도 있습니다.

---

### 2. 코드 블록 (Code Blocks)
코드의 문법 강조(Syntax Highlighting)를 위해 사용합니다. 숫자 1 옆의 `` ` ``(백틱) 기호를 사용합니다.

**[입력]**
` ``` ` 뒤에 언어 이름(python 등)을 쓰면 색상이 적용됩니다.

<pre>
```python
def hello():
    print("Hello, Markdown!")
```
</pre>

**[결과]**
```python
def hello():
    print("Hello, Markdown!")
```

*   **인라인 코드:** 문장 중간에 코드를 넣을 때는 백틱 하나(`` ` ``)로 감쌉니다.
    *   예: 변수명은 `variable` 입니다.

---

### 3. 링크와 이미지 (Links & Images)
외부 사이트로 연결하거나 이미지를 첨부합니다.

**[입력]**
```markdown
[구글 바로가기](https://google.com)

![이미지 설명(엑박뜰때 보임)](https://www.python.org/static/img/python-logo.png)
```

**[결과]**
*   [구글 바로가기](https://google.com)
*   (파이썬 로고 이미지가 출력됨)

> **팁:** 이미지는 `!`(느낌표)가 앞에 붙습니다. "링크는 **Link**라서 `[]`, 이미지는 **Image**라서 `!`"라고 기억하면 쉽습니다.

---

### 4. 구분선 (Horizontal Rule)
내용을 분리할 때 가로 선을 긋습니다.

**[입력]**
```markdown
---
(또는 ***)
```

**[결과]**
(가로줄이 생성됩니다)
***

---

### 5. 표 만들기 (Tables) - **중급 핵심**
데이터를 표로 정리할 때 사용합니다. `|`(파이프)와 `-`(대시)를 조합합니다.
콜론(`:`)의 위치에 따라 정렬이 바뀝니다.

**[입력]**
```markdown
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| :--- | :---: | ---: |
| 내용 1 | 내용 2 | 내용 3 |
| Left | Center | Right |
```

**[결과]**

| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| :--- | :---: | ---: |
| 내용 1 | 내용 2 | 내용 3 |
| Left | Center | Right |

---

### (보너스) 수식 입력 (LaTeX)
쥬피터 노트북은 수학 공식 입력도 지원합니다. 달러 기호(`$`)를 사용합니다.

*   **문장 내 수식:** `$y = ax + b$` → $y = ax + b$
*   **독립된 수식:** `$$E = mc^2$$` → (가운데 정렬되어 크게 표시됨)

# 🚀 쥬피터 노트북 마크다운 가이드 (고급: HTML 활용)

기본 마크다운 문법만으로는 글자 색상을 바꾸거나, 이미지 크기를 조절하거나, 내용을 접었다 펴는 기능을 구현할 수 없습니다. 이때는 HTML 코드를 직접 사용합니다.

> **⚠️ 주의:** 깃허브(GitHub) 등 일부 플랫폼에서는 보안상의 이유로 아래의 HTML 태그(특히 style 속성)가 적용되지 않고 원본 텍스트만 보일 수 있습니다. 하지만 **Jupyter Notebook, JupyterLab, Google Colab** 환경에서는 잘 작동합니다.

## 🔴 1. 텍스트 색상 및 배경색 (Colors)

가장 많이 질문하는 기능입니다. HTML5 표준은 `<span>` 태그를 권장하지만, 노트북 환경에서는 옛날 방식인 `<font>` 태그가 타이핑이 빠르고 렌더링 호환성이 좋을 때가 많습니다.

### 1-1. 글자 색상 변경 (`<font>`)
가장 간단하게 색을 입히는 방법입니다.

**[입력]**
```html
<font color="red">빨간색 텍스트</font>입니다.
<font color="#0000FF">파란색(Hex Code)</font>입니다.
<font color="green" size="5">크고 초록색인 텍스트</font>
```

**[결과]**
<font color="red">빨간색 텍스트</font>입니다.
<font color="#0000FF">파란색(Hex Code)</font>입니다.
<font color="green" size="5">크고 초록색인 텍스트</font>

---

### 1-2. 텍스트 배경색 (형광펜 효과)
배경색은 `<font>` 태그로 안 되므로 `<span>`이나 `<mark>` 태그를 사용해야 합니다.

**[입력]**
```html
일반 텍스트와 <span style="background-color: yellow; color: black;">노란 형광펜</span> 효과
또는 <mark>마크 태그(기본 노랑)</mark> 사용하기
```

**[결과]**
일반 텍스트와 <span style="background-color: yellow; color: black;">노란 형광펜</span> 효과
또는 <mark>마크 태그(기본 노랑)</mark> 사용하기

---

## 🖼️ 2. 이미지 크기 조절 및 정렬

마크다운 `![]()` 문법은 이미지 크기 조절이 불가능합니다. HTML `<img>` 태그를 사용하면 해결됩니다.

**[입력]**
```html
<!-- 크기 조절 -->
<img src="https://www.python.org/static/img/python-logo.png" width="200" height="60">

<!-- 왼쪽/오른쪽 정렬 (align 속성 사용) -->
<div align="center">
  <img src="https://www.python.org/static/img/python-logo.png" width="100">
  <p>가운데 정렬된 이미지와 캡션</p>
</div>
```

**[결과]**
(가로 200px로 줄어든 이미지와, 화면 가운데 정렬된 이미지가 출력됨)

## 📦 3. 알림 박스 (Colored Boxes)

강의 자료에서 **"참고(Note)", "경고(Warning)"** 등을 표현할 때 매우 유용합니다. `<div>` 태그에 스타일을 입혀 만듭니다.

**[입력]**
```html
<!-- 파란색 정보 박스 -->
<div style="background-color: #e6f7ff; border-left: 5px solid #1890ff; padding: 10px;">
  <b>ℹ️ Info:</b> 여기에 중요한 정보를 입력하세요.
</div>

<br>

<!-- 붉은색 경고 박스 -->
<div style="background-color: #fff1f0; border: 1px solid #ffa39e; padding: 10px; border-radius: 5px;">
  <b>🚨 Warning:</b> 데이터를 삭제하면 복구할 수 없습니다.
</div>
```

**[결과]**
(파란색 배경의 깔끔한 박스와 붉은색 경고 박스가 생성됨)

> **Tip:** 코랩이나 쥬피터 랩에서는 위 코드가 아주 예쁘게 렌더링 됩니다.

---

## 🔽 4. 접기/펼치기 (Collapsible Content)

정답 코드나 너무 긴 로그(Log)를 숨겨두고 싶을 때 사용합니다. `<details>`와 `<summary>` 태그를 씁니다.

**[입력]**
```html
<details>
<summary>👉 정답 확인하기 (클릭)</summary>
    
<br> <!-- 줄바꿈을 넣어줘야 내부 마크다운이 잘 먹힙니다 -->

print("이것이 정답 코드입니다.")

</details>
```

**[결과]**
(클릭하면 내용이 펼쳐지는 토글 버튼 생성)

---

## 🔗 5. 페이지 내 이동 (Anchor)

문서가 길어질 때 목차(TOC)를 만들고 특정 위치로 점프하게 만듭니다.

**1. 목적지에 닻(id) 내리기:**
이동하고 싶은 곳(제목 등) 바로 위에 아래 코드를 씁니다.
```html
<a id="my_destination"></a>
### 여기로 이동합니다
```

**2. 링크 걸기:**
```markdown
[이동하기 버튼](#my_destination)
```

| 기능 | 태그 예시 (쥬피터 친화적) |
| :--- | :--- |
| **글자 색** | `<font color="red">Text</font>` |

| **접기/펼치기** | `<details><summary>제목</summary>내용</details>` |
| **공백(띄어쓰기)** | `&nbsp;` (여러 번 쓰면 강제로 띄워짐) |
