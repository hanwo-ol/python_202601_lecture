# 정답 및 해설 (검증용 코드)

아래 버튼을 클릭하여 정답과 실제 작동 코드를 확인하세요.

<details>
<summary>1번 ~ 5번 정답 및 해설</summary>

<br>

**1. 정답: O**
*   **해설:** 파이썬 3 버전부터는 `/` 연산자는 항상 실수(float)를 반환합니다. 딱 떨어지는 수라도 `3.0`과 같이 표현됩니다.
*   **검증 코드:**
    ```python
    print(4 / 2)       # 2.0 출력
    print(type(4 / 2)) # <class 'float'> 출력
    ```

**2. 정답: X**
*   **해설:** `//`는 나눗셈의 **몫(소수점 버림)**을 구하는 연산자입니다. 나머지를 구하는 연산자는 `%`입니다.
*   **검증 코드:**
    ```python
    print(7 // 3) # 2 (몫)
    print(7 % 3)  # 1 (나머지)
    ```

**3. 정답: X**
*   **해설:** `**`는 거듭제곱 연산자입니다. `2의 3승(2 * 2 * 2)`이므로 결과는 `8`입니다.
*   **검증 코드:**
    ```python
    print(2 ** 3) # 8
    ```

**4. 정답: X**
*   **해설:** 연산자 우선순위에 의해 곱셈(`*`)이 덧셈(`+`)보다 먼저 계산됩니다. `3 * 4`를 먼저 하여 `12`가 되고, 여기에 `2`를 더하므로 결과는 `14`입니다.
*   **검증 코드:**
    ```python
    print(2 + 3 * 4) # 14
    ```

**5. 정답: X**
*   **해설:** `=`는 변수에 값을 저장하는 **할당(대입) 연산자**입니다. '같다'를 비교할 때는 `==`를 사용해야 합니다.
*   **검증 코드:**
    ```python
    x = 10     # x에 10을 할당
    print(x == 10) # x가 10과 같은가? (True)
    ```

</details>

<details>
<summary>6번 ~ 10번 정답 및 해설</summary>

<br>

**6. 정답: O**
*   **해설:** `!=`는 'Not Equal'의 의미로, 두 값이 서로 다르면 참(True)을 반환합니다.
*   **검증 코드:**
    ```python
    print(10 != 5) # True
    ```

**7. 정답: O**
*   **해설:** 파이썬에서 문자열에 정수를 곱하면 해당 횟수만큼 문자열이 반복됩니다.
*   **검증 코드:**
    ```python
    print("Hello" * 3) # HelloHelloHello
    ```

**8. 정답: X**
*   **해설:** 숫자(int)와 문자열(str)은 서로 성격이 달라 직접 더할 수 없습니다. `TypeError`가 발생합니다. 형 변환(casting)을 해야 합니다.
*   **검증 코드:**
    ```python
    # print(100 + "원") # 에러 발생
    print(str(100) + "원") # 100원 (정상)
    ```

**9. 정답: X**
*   **해설:** `and` 연산자는 양쪽 조건이 **모두 True**일 때만 True를 반환합니다. 하나라도 False면 결과는 False입니다.
*   **검증 코드:**
    ```python
    print(True and False) # False
    ```

**10. 정답: O**
*   **해설:** `or` 연산자는 둘 중 **하나라도 True**면 True를 반환합니다. `10 > 5`가 True이므로 뒤의 조건과 상관없이 결과는 True입니다.
*   **검증 코드:**
    ```python
    print(10 > 5 or 5 > 10) # True
    ```

</details>

**11. 정답: O**
*   **해설:** `10 == 10`은 True입니다. 여기에 `not`을 붙이면 반대인 False가 됩니다.
*   **검증 코드:**
    ```python
    print(not (10 == 10)) # False
    ```

**12. 정답: O**
*   **해설:** `+=`는 복합 할당 연산자로, 기존 변수 값에 값을 더하여 다시 저장하는 문법입니다.
*   **검증 코드:**
    ```python
    a = 5
    a += 10
    print(a) # 15
    ```

**13. 정답: O**
*   **해설:** `%`는 나머지를 구합니다. 10을 3으로 나누면 몫은 3이고 나머지는 1입니다.
*   **검증 코드:**
    ```python
    print(10 % 3) # 1
    ```

**14. 정답: O**
*   **해설:** 파이썬의 강력한 기능 중 하나로, 수학 부등식처럼 연속된 비교가 가능합니다. 다른 많은 언어에서는 지원하지 않는 기능입니다.
*   **검증 코드:**
    ```python
    x = 15
    print(10 < x < 20) # True
    ```
