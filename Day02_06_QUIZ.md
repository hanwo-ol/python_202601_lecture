# 파이썬 연산자 마스터 OX 퀴즈 (https://buly.kr/ESzlR9h)

다음 문장이 맞으면 **O**, 틀리면 **X**를 선택하세요. 코랩을 이용해서 실습을 하고 답변을 다셔도 됩니다.

1. 파이썬에서 나눗셈 연산자 `/`의 결과는 정수끼리 나누더라도 항상 **실수(float)** 형태이다. ( )
2. 연산자 `//`는 나눗셈의 **나머지**를 구하는 연산자이다. ( )
3. `2 ** 3`의 계산 결과는 `6`이다. ( )
4. 수식 `2 + 3 * 4`의 계산 결과는 `20`이다. ( )
5. 수학에서 '같다'는 의미인 `=` 기호는 파이썬에서 **비교 연산자**로 사용된다. ( )
6. `!=` 연산자는 두 값이 **다를 때** `True`를 반환한다. ( )
7. 문자열에도 산술 연산자를 사용할 수 있으며, `"Hello" * 3`은 에러가 발생하지 않는다. ( )
8. 정수와 문자열은 더하기 연산이 가능하다. (예: `100 + "원"`) ( )
9. `True and False`의 결과는 `True`이다. ( )
10. `10 > 5 or 5 > 10`의 결과는 `True`이다. ( )
11. `not (10 == 10)`의 결과는 `False`이다. ( )
12. `a += 10`은 `a = a + 10`과 같은 의미이다. ( )
13. `10 % 3`의 결과 값은 `1`이다. ( )
14. 파이썬에서는 `10 < x < 20`과 같이 부등호를 연속해서 작성할 수 있다. ( )
15. 어떤 수를 `0`으로 나누면 결과는 `0` 또는 `Infinity`가 된다. ( )
